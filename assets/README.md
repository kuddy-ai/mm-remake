# Pixel Art Resource Layout

The demo uses a 16x16 tile grid and nearest-neighbor scaling.

- `terrain/`: sand, rock, wasteland dirt, scrap metal, ruin wall tiles
- `character/`: small hunter and NPC sprite sheets
- `vehicle/`: tank bodies, turrets, tracks, rust decals
- `monster/`: mechanical dog, mutant bug, scrap drone sprites
- `ui/`: pixel panels, bars, buttons, inventory slots, dialog frames
- `background/`: tiled mountains, ruins, dust sky props

Current prototype art is generated procedurally in `game/demo_main.gd` with the same density and categories.
