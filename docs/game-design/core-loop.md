# 荒原战车 REMAKE v1

当前 v1 使用 `scenes/main.tscn` 作为启动场景，运行项目后会进入完整的垂直切片流程。

## 页面

- 启动动画：展示项目 Logo、扫描线和进入标题的转场。
- 标题界面：展示新游戏入口和任务档案。
- 新手村：在锈镇维修站移动，和修理师马库斯交谈，前往西侧废弃加油站。
- 第一场战斗：与变异鬣狗进行回合制战斗，支持主炮、手枪、急救包和防御。
- 胜利结算：展示新手流程完成和奖励。

## 音乐

v1 使用 Godot 的 `AudioStreamGenerator` 程序生成背景音乐，不依赖外部音频文件。标题、新手村、战斗和胜利状态会切换不同节奏与音色。

## 操作

- 鼠标点击按钮选择菜单和战斗指令。
- 方向键或 WASD 在新手村移动。
- 空格或 Enter 与 NPC / 场景交互。
- 战斗中空格默认触发主炮攻击。

## 运行

```bash
./scripts/run.sh
```

或者用 Godot 4.x 打开 `project.godot` 后运行项目。
