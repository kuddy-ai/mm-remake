# Wasteland Hunter Curated Pixel Design Pack


本包只保留当前确定的方向：横板像素风、废土放置 RPG、城镇 Hub、路线探索、自动战斗、车库升级和系统 UI。

## Included Files

- `01_style/01_character_tank_pixel_style_sheet.webp` — 主角、战车、动画帧、基础场景的像素风规格参考
- `01_style/02_side_scrolling_pixel_scene_reference.webp` — 横板像素场景基准：主角、战车、废土远景、基础 HUD
- `02_core_screens/01_title_screen_main_menu.webp` — 游戏标题页 / 主菜单 / 放置收益入口
- `02_core_screens/02_rustwalk_town_hub_nodes.webp` — 横板城镇 Hub：建筑入口、任务提示、底部导航
- `02_core_screens/03_settlement_management_hub.webp` — 城镇经营型 Hub：车库、酒馆、商店、医疗室、任务面板
- `02_core_screens/04_chapter_route_map_screen.webp` — 章节路线图：关卡节点、奖励、队伍选择、开始路线
- `02_core_screens/05_wasteland_road_exploration_screen.webp` — 路线探索：事件节点、燃料、自动旅行、随机事件
- `03_battle/01_idle_auto_battle_main_layout.webp` — 放置自动战斗主界面：资源栏、队伍、敌人、收益
- `03_battle/02_side_scrolling_combat_clean_layout.webp` — 横板战斗简洁版布局：角色、战车、敌方载具、底部技能栏
- `03_battle/03_iron_colossus_boss_battle.webp` — 大型 Boss 自动战斗：弱点、奖励、技能、Boss 血条
- `03_battle/04_underground_facility_boss_battle.webp` — 地下设施战斗：旧世界设施、机械 Boss、怪物组合
- `03_battle/05_victory_rewards_summary.webp` — 战斗胜利结算：星级、掉落、经验、放置收益
- `04_systems/01_garage_tank_upgrade_interface.webp` — 车库 / 战车升级：部件、属性、碎片、升级成本
- `04_systems/02_scrap_shop_store_interface.webp` — 废品商店：零件、武器、消耗品、购买/出售
- `04_systems/03_tavern_bounty_recruit_interface.webp` — 酒馆 / 赏金任务 / 招募界面
- `04_systems/04_hunter_equipment_inventory_interface.webp` — 角色装备、背包、技能与属性界面
- `04_systems/05_barracks_team_formation_interface.webp` — 兵营 / 队伍编成 / 前排支援与车辆站位
- `04_systems/06_radio_tower_dispatch_interface.webp` — 通讯塔 / 派遣 / 地图信号与任务队列
- `04_systems/07_medical_bay_treatment_interface.webp` — 医疗室：治疗队列、状态异常、药品消耗
- `04_systems/08_bounty_bestiary_dossier_interface.webp` — 赏金图鉴 / 怪物档案 / 弱点与掉落
- `05_environment/01_rustwalk_settlement_side_view.webp` — 横板城镇外观：酒馆、车库、城门与地下区域
- `06_reference_boards/01_phase2_reference_board_cn.webp` — 阶段 2 扩展参考：矿洞、军事基地、废弃城市、NPC、怪物、地图与系统界面
