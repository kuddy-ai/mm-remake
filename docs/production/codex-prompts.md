# Codex Prompts

## 场景实现提示词

实现 UI 场景时，请参考：

```
请参考 docs/ui-design/ui-style-guide.md 和 docs/concept-art/wasteland_hunter_design_images/
实现 [具体 UI 场景名称]，使用 UIStyleGuide 类的颜色和尺寸常量。
```

## 美术实现提示词

```
请参考 docs/art-direction/pixel-style-guide.md

```
